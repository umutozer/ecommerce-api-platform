# ECommerce API Platform - Frontend

## Tech Stack
- React 18
- Vite
- React Router DOM
- Tailwind CSS
- Axios

## Setup

```bash
npm install
npm run dev
```

## Features
- User authentication (login/register)
- Product management
- Category management
- Order tracking
- API documentation viewer
- Responsive design

## Environment

Make sure the backend API is running on `http://localhost:5000`

## Build

```bash
npm run build
```
